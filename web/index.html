CSV 메타데이터 간격 분석

신규 Tailing CSV: 1행 hub_id,device_mac,start_time,end_time → 2행 세션 값(start_time은 YYMMDD-HH:mm:ss:SSS, KST) → 3행 timestamp,ir,red,green,hr,spo2,temp,battery,gyro → 이후 데이터. 데이터 timestamp는 시각만 있으며 자정 넘김 시 일 단위로 자동 증가해 해석합니다.
갭(배치): 실측 vitals가 있는 행(숫자 HR·spo2·temp 등)을 앵커로 하되, Poor Signal만 있고 spo2/temp가 비어 있으면 앵커에서 제외합니다.
PPG 샘플링: 연속 행 timestamp 간격(ms) 통계로 샘플 주기를 디버깅합니다. 구형(단일 헤더·행마다 hub) CSV도 그대로 지원합니다. 쉼표/탭 모두 가능합니다.

CSV 파일을 여기에 놓거나 클릭하여 선택
여러 파일 동시 선택 가능 · *_vitals.csv는 건너뜀
날짜 비우면 파일명의 YYYY-MM-DD 또는 YYYYMMDD 사용.
